RFC 8997
Deprecation of TLS 1.1 for Email Submission and
Access

Abstract

This speciﬁcation updates the current recommendation for the use of the Transport Layer
Security (TLS) protocol to provide conﬁdentiality of email between a Mail User Agent (MUA) and
a Mail Submission Server or Mail Access Server. This document updates RFC 8314.
