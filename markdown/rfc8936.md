RFC 8936
Poll-Based Security Event Token (SET) Delivery Using
HTTP

Abstract

This speciﬁcation deﬁnes how a series of Security Event Tokens (SETs) can be delivered to an
intended recipient using HTTP POST over TLS initiated as a poll by the recipient. The
speciﬁcation also deﬁnes how delivery can be assured, subject to the SET Recipient's need for
assurance.
