Abstract

RFC 8287 deﬁnes the extensions to perform LSP Ping and Traceroute for Segment Routing IGP-
Preﬁx and IGP-Adjacency Segment Identiﬁers (SIDs) with the MPLS data plane. RFC 8287
proposes three Target Forwarding Equivalence Class (FEC) Stack sub-TLVs. While RFC 8287
deﬁnes the format and procedure to handle those sub-TLVs, it does not suﬃciently clarify how
the length of the Segment ID sub-TLVs should be computed to be included in the Length ﬁeld of
the sub-TLVs. This ambiguity has resulted in interoperability issues.

This document updates RFC 8287 by clarifying the length of each of the Segment ID sub-TLVs
deﬁned in RFC 8287.
