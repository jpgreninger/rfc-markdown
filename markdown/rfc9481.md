RFC 9481
Certificate Management Protocol (CMP) Algorithms

Abstract

This document describes the conventions for using several cryptographic algorithms with the
Certificate Management Protocol (CMP). CMP is used to enroll and further manage the lifecycle of
X.509 certificates. This document also updates the algorithm use profile from Appendix D.2 of
RFC 4210.
