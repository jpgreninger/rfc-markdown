RFC 8951
Clariﬁcation of Enrollment over Secure Transport
(EST): Transfer Encodings and ASN.1

Abstract

This document updates RFC 7030: Enrollment over Secure Transport to resolve some errata that
were reported and that have proven to cause interoperability issues when RFC 7030 was
extended.

This document deprecates the speciﬁcation of "Content-Transfer-Encoding" headers for
Enrollment over Secure Transport (EST) endpoints. This document ﬁxes some syntactical errors
in ASN.1 that were present.
