RFC 9220
Bootstrapping WebSockets with HTTP/3

Abstract

The mechanism for running the WebSocket Protocol over a single stream of an HTTP/2
connection is equally applicable to HTTP/3, but the HTTP-version-speciﬁc details need to be
speciﬁed. This document describes how the mechanism is adapted for HTTP/3.
