Abstract

This document updates and corrects the IANA "XML Security URIs" registry that lists URIs
intended for use with XML digital signatures, encryption, canonicalization, and key
management. These URIs identify algorithms and types of information. This document also
obsoletes and corrects three errata against RFC 6931.

Implementers should be aware that cryptographic algorithms become weaker with time. As new
