RFC 9093
A YANG Data Model for Layer 0 Types

Abstract

This document deﬁnes a collection of common data types and groupings in the YANG data
modeling language. These derived common types and groupings are intended to be imported by
modules that model Layer 0 optical Traﬃc Engineering (TE) conﬁguration and state capabilities
such as Wavelength Switched Optical Networks (WSONs) and ﬂexi-grid Dense Wavelength
Division Multiplexing (DWDM) networks.
