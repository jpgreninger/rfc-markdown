RFC 8709
Ed25519 and Ed448 Public Key Algorithms for the
Secure Shell (SSH) Protocol

Abstract

This document describes the use of the Ed25519 and Ed448 digital signature algorithms in the
Secure Shell (SSH) protocol. Accordingly, this RFC updates RFC 4253.

string

signature

.
